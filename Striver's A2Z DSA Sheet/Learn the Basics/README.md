# Learn the Basics

This section contains my solutions and practice exercises from the "Learn the Basics" part of Striver's A2Z DSA Sheet.

The goal of this section is to build a strong programming foundation before moving on to advanced Data Structures and Algorithms topics.

Topics covered include:

* Things to Know in C++ / Programming Fundamentals
* Logical Thinking and Pattern Problems
* Basic Mathematics
* Basic Recursion
* Basic Hashing

Through these problems, I focus on understanding problem-solving techniques, writing clean code, and analyzing time and space complexity.

Language Used: C++
